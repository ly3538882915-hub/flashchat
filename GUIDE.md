# FlashChat V0.2 — 让朋友一起使用指南

## 方案一：同一 WiFi/局域网内使用（最简单）

如果你的朋友和你连着同一个 WiFi（比如在家里、办公室），这是最简单的方式。

### 第一步：启动服务器

打开终端（命令提示符），运行：
```
cd F:\TelegramCN\flash_chat_web
node server.js
```

启动后你会看到类似这样的输出：
```
========================================
  FlashChat Web v0.2 已启动
========================================
  本机访问:   http://localhost:3000
  局域网访问: http://192.168.x.x:3000
========================================
```

**记住那个「局域网访问」地址**，这就是你朋友要用的地址。

### 第二步：告诉你的朋友

把那个地址（比如 `http://192.168.1.100:3000`）发给你的朋友。

你的朋友只需要：
1. 在手机或电脑浏览器里打开这个地址
2. 注册一个账号
3. 就能用了！

### 第三步：互加好友开始聊天

1. 你和朋友都注册登录后
2. 点右上角 **"+"** 按钮
3. 搜索对方的用户名
4. 点击搜索结果，开始聊天！

### 注意事项
- **你的电脑必须保持开机**，服务器关了朋友就连不上了
- **必须连同一个 WiFi**（比如你家路由器的 WiFi）
- 如果连不上，可能是 Windows 防火墙拦截了，需要放行 3000 端口

### Windows 防火墙放行方法
1. 打开「控制面板」→「Windows Defender 防火墙」→「高级设置」
2. 左侧点「入站规则」→ 右侧点「新建规则」
3. 选「端口」→ TCP → 特定端口输入 `3000` → 允许连接 → 完成

## 方案二：让远方的朋友也能用（内网穿透）

如果朋友不在你身边，不在同一个 WiFi 下，可以用**内网穿透**工具，把你电脑的服务暴露到公网。

### 推荐：使用 ngrok（免费，5 分钟搞定）

#### 1. 下载 ngrok
- 去 https://ngrok.com 注册并下载
- 解压后得到 `ngrok.exe`

#### 2. 配置 authtoken
```
ngrok config add-authtoken 你的token
```
（注册后在 ngrok 网站的 Dashboard 能看到你的 token）

#### 3. 启动隧道
确保 FlashChat 服务在跑着，然后另开一个终端：
```
ngrok http 3000
```

你会看到类似输出：
```
Session Status    online
Forwarding        https://abc-123-456.ngrok-free.app -> http://localhost:3000

Connections       ttl     opn     rt1     ...
```

#### 4. 把那个 `https://abc-123-456.ngrok-free.app` 地址发给朋友
朋友在任何地方打开这个地址就能用了！手机、电脑都行。

### 替代方案：Cloudflare Tunnel（也是免费的）
```
cloudflared tunnel --url http://localhost:3000
```
会给你一个 `https://xxx.trycloudflare.com` 的地址，用法一样。

## 方案三：部署到云服务器（长期稳定运行）

如果你想让聊天软件 24 小时在线，不用每次都开电脑，可以部署到云服务器。

### 推荐：Railway / Render（免费额度）

1. 把 `flash_chat_web` 文件夹上传到 GitHub
2. 在 Railway.app 或 Render.com 创建新项目，连接 GitHub 仓库
3. 设置启动命令为 `node server.js`
4. 自动部署，获得一个公网地址
5. 把地址发给朋友

### 注意
- SQLite 数据存在容器里，重启会丢失（如需持久化要换 PostgreSQL 或挂载卷）
- 免费额度有访问量限制

## 常见问题

**Q: 朋友打开页面显示"无法访问"？**
A: 检查：1) 你的服务器在跑着 2) 防火墙放行了 3000 端口 3) 你们在同一个 WiFi

**Q: 朋友能打开页面但注册不了？**
A: 不会发生，注册是前端+后端一起的，能打开页面就能注册

**Q: 数据安全吗？**
A: 所有数据存在你电脑的 `data/chat.db` 文件里，密码已加密。你不分享数据库文件，别人拿不到数据

**Q: 能同时多少人用？**
A: FlashChat V0.2 用 Node.js + SQLite，同时 50-100 人聊天没问题。更多人需要升级数据库

**Q: 朋友用手机打开体验怎么样？**
A: V0.2 做了完整的移动端适配，手机浏览器打开就是 Telegram 风格的界面
